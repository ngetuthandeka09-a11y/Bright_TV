# Bright_TV

*Project Overview*
This project analyses viewers sessions from BrightTV's South African subscriber base using two relational datasets. The goal was to extract meaningful insights on user behaviour, consumption patterns and growth opportunities,using SQL, data visualisation and storytelling.


📊 *Key Findings*
👤 Gender81.5% male — female audience largely untapped at 9.3%
🏆 Top Province Gauteng drives 34.1% of all sessions
📡 #1 ChannelSuperSport Live Events — 1,638 sessions
📅 Busiest DayFriday — 1,675 sessions
📉 Quietest DayMonday — only 957 sessions
⏰ Peak ViewingEvening (18–22h) — 27.1% of all sessions
⏱ Session Length 63.5% of sessions under 5 minutes
🌍 Top 3 ProvincesGauteng + Western Cape + KZN = 58.5% of views

💡 *Recommendations*

Female content strategy — Only 9.3% female viewers; commission female-led programming
Monday activation — Premieres & sports replays to lift the week's lowest day
Rural expansion — Limpopo, Northern Cape & North West remain under-served
Long-form content — 63.5% of sessions are <5 min; invest in binge-worthy series
Evening prime slot — 18–22h is peak; prioritise premium content here
Local language programming — Zulu, Xhosa & Sotho content for broader reach
